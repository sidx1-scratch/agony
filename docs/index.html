Instruction Decoder

Every single instruction requires five simultaneous encodings. Change any one and the whole thing breaks.

Step 1: Binary Opcode
00001000
The raw byte
8-bit representation. Immutable. No shortcuts.
↓
Step 2: Assembly Mnemonic
MOV
Must match the binary
Opcode 8 = MOV. Any mismatch = compilation failure.
↓
Step 3: Operands (Emoji-Tagged)
📍R0🔢5
Each prefixed with its type emoji
📍=register, 🔢=number, 🔤=string, 🎯=label. No spaces.
↓
Step 4: Parity Check
❌
Computed by hand. Correct or fail.
00001000 has 1 set bit (odd) → ❌. Even parity → ✅
↓
Step 5: Checksum (Decimal as Emoji)
8️⃣
Opcode value in keycap digits
Opcode 8 → 8️⃣. Opcode 14 → 1️⃣4️⃣. Triple redundancy.
↓
Final Flourish
🏁
Mandatory. Non-negotiable.
Every instruction ends with this or it refuses to compile.
Complete instruction:
00001000::MOV::📍R0🔢5::❌::8️⃣::🏁

What You Get

8 Registers (R0–R7)

Full working register set. Your own state machine waiting to be tortured.

Stack Operations

PUSH, POP. All the classic ways to lose track of what you put where.

Arithmetic

ADD, SUB, MUL, DIV. Works exactly as you'd expect, no surprises.

Control Flow

JMP, JZ, JNZ. Unconditional and conditional jumps. Loops are technically possible.

I/O

PRINT (integers), PRINTS (strings). Get your output. Doubt it.

Zero Dependencies

Single C++ file. One compiler. No frameworks, no build systems. Just suffering.

Opcode Reference

00000001 :: NOP — do nothing
00000010 :: PUSH — push value to stack
00001000 :: MOV — dest = src
00000100 :: ADD — dest += src
00000101 :: SUB — dest -= src
00001001 :: PRINT — print integer
00001010 :: PRINTS — print string
00001011 :: JMP — unconditional jump
00001110 :: JNZ — jump if not zero
00001111 :: HALT — stop execution
📍 = register (R0–R7)
🔢 = number literal (e.g., 5, -42)
🔤 = string literal ("quoted text")
🎯 = label (bare identifier like loop, start)
✅ = even parity | ❌ = odd parity
Keycap digits = opcode decimal value (8️⃣, 1️⃣4️⃣)

Quick Start

  1. Download the latest release from GitHub or clone the repo.
  2. Extract: unzip agony.zip && cd agony
  3. Run the installer: ./install.sh
  4. Try an example: agony hello.agony
  5. Spend the next 3 hours hand-computing bit parity and emoji checksums.

Manual build: g++ -std=c++17 -O2 -o agony agony.cpp then ./agony program.agony
